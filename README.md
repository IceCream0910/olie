# Introduction
멜론과 유튜브 API를 활용한 무료 스트리밍 서버입니다.  
최소한의 기능에 충실하되 프로그램 중심의 스트리밍 서비스들을  
대체할 수 있는 개인용 서비스를 제작하고자 했습니다.  

# Function
* 차트 파싱 기능(멜론 사이트 크롤링)
* 음악 재생 및 플레이리스트 관리 기능 
* 검색 기능

# How to build
> python 3.6 이상 버전
* 사용한 라이브러리
  * Flask 
  * BeautifulSoup 
  * requests  

정상적인 사용을 위해 script.js에 유튜브 api키를 할당해줘야합니다.
